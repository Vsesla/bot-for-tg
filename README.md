
# TeaDynasty — сайт для продажу китайського чаю

Готовий сайт-магазин китайського чаю з каталогом, кошиком і оформленням замовлення.

## Що реалізовано
- Каталог товарів із пошуком і фільтрацією за категоріями.
- Додавання товарів у кошик, зміна кількості, автоматичний підрахунок суми.
- Форма оформлення замовлення (ім'я, телефон, місто, адреса, спосіб оплати).
- Збереження кошика та замовлень у `localStorage` браузера.
- Адаптивний UI для мобільних і десктопних пристроїв.

## Запуск
```bash
python -m http.server 8000
```

Відкрийте у браузері: `http://127.0.0.1:8000`.
